Individual Learning Plans

How to Generate an Individual Learning Plan using ChatGPT

  1. Open ChatGPT — Click the button below to open ChatGPT in a new tab, or go to chat.openai.com
  2. Gather all relevant documentation for the student, including:
    • Rapid response information
    • Termly reports
    • Mentoring sessions
    • Student passports
    • Any other relevant information about the student
  3. Copy the prompt below — Use the "Copy Prompt" button to copy the ILP generation prompt
  4. Paste documentation and prompt into ChatGPT — First paste all the gathered documentation, then paste the prompt below it
  5. Review and refine — Review the generated ILP and ask ChatGPT to make any necessary adjustments or additions

Information to Include

When gathering documentation to paste into ChatGPT, make sure to include:

  • Rapid Response: Initial assessments, immediate concerns, urgent support needs
  • Termly Reports: Academic progress, attendance, behaviour, SEMH progress, engagement, achievements
  • Mentoring Sessions: Notes from mentoring sessions, progress updates, targets discussed
  • Student Passports: Key information about the student's needs, preferences, and background
  • Other Relevant Information: Parent/carer feedback, external agency involvement, previous ILP outcomes, specific learning needs, interests, strengths, barriers to learning
